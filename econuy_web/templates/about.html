Sobre econuy

econuy facilita el acceso a estadísticas de la economía uruguaya. Apunta a:

  • Reunir datos de varios sitios web en el mismo lugar (INE, BCU, MEF, FMI, etc.).
  • Presentar planillas limpias, sin notas, sin títulos, en formato tidy data.
  • Incluir metadatos claros para cada indicador.
  • Permitir el cálculo de transformaciones sobre los datos (por ejemplo, convertir a dólares).
  • Visualizar los datos.
  • Permitir descargar los datos.

Dos veces al día se actualizan todos los datos disponibles, por lo que el tiempo entre que nueva información se hace pública y están disponibles en econuy debería ser bastante corto.

Los datos son descargados automáticamente de sitios públicos, lo que significa que puede haber demoras en las actualizaciones cuando a) la URL desde la que se descargan los datos cambia; b) la URL no está disponible; c) el archivo de origen cambió de manera no esperada. En ese caso, se puede crear un issue en Github para alertarme del problema.

En paralelo a esta web, econuy puede ser usado como un paquete de Python. Las instrucciones para esto se encuentran en el README.
